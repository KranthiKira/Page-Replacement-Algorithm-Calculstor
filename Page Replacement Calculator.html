Introduction

Page Fault:

When the page referenced by the CPU is not found in the main memory, the situation is called Page Fault.

Page Replacement Algorithms:

Page replacement algorithms are used to decrease the maximum number of page faults.

Instructions

  2. Select page replacement algorithm
